# Description
Simple plugin of JQuery to select all checkbox.

#Install

In a browser:

    <script src="yourpath/jquery.min.js"></script>
    <script src="yourpath/jquery.selectAll.js"></script>
    
#Usage

Simple Use:

     $.selectAll();
     
or

    $.selectAll({
        selectbutton : '.j-selectAll', 
        checkbox : '.j-chk' 
    });
